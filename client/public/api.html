API publique

Une API REST en lecture seule pour explorer les chroniques, séries et mangaka de Tsundoku. Réponses JSON, sans authentification.

MéthodeEndpointDescription
GET/api/postsListe paginée des chroniques (params : page, limit, category, tag, type, sort, rating_min)
GET/api/posts/featuredChronique mise en avant
GET/api/posts/randomUne chronique au hasard
GET/api/posts/search?q=Recherche plein-texte
GET/api/posts/:slugUne chronique + articles adjacents
GET/api/posts/:slug/commentsCommentaires approuvés
GET/api/booksSéries (params : status, source, category, q, sort)
GET/api/books/:slugFiche série + planches + similaires
GET/api/authorsMangaka
GET/api/authors/:slugMangaka + bibliographie + stats
GET/api/categoriesGenres
GET/api/tagsTags
GET/api/lists · /api/lists/:slugListes thématiques
GET/api/stats/publicStatistiques publiques
GET/api/timelineJournal de lecture
GET/rss.xml · /sitemap.xmlFlux RSS · Sitemap
